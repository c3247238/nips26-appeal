# 方法论者视角

## 实验设计评估

### 优点
1. **组件隔离设计**：每次只改变一个架构元素，因果归因清晰
2. **对照充分**：Random control验证了指标区分度
3. **多seed设计**：5个seed提供基础统计可靠性
4. **数据完整性检查**：MD5哈希检测、特征数验证、输出文件检查

### 缺陷
1. **缺少L0匹配**：这是方法学上最关键的缺失。没有L0匹配，架构比较是 apples-to-oranges
2. **收敛诊断缺失**：训练时间异常短（2-3秒），缺少loss曲线验证
3. **统计方法不完整**：未做显著性检验、效应量计算、多重比较校正
4. **样本量不足**：5个seed仅能提供初步证据，不足以支撑强结论

### 建议改进
1. **预注册分析计划**：在查看数据前定义假设、检验方法、决策规则
2. **增加样本量**：至少10个seed，或考虑bootstrap置信区间
3. **添加收敛诊断**：每个训练运行必须包含loss曲线和最终loss值
4. **标准化Cohen's d**：统一使用pooled std，在方法节明确声明公式

### 可信度评估
| 声明 | 当前可信度 | 需要做什么才能提高 |
|------|-----------|-------------------|
| TopK吸收率低 | 低（L0混淆） | L0匹配实验 |
| OrtSAE无效果 | 中 | L0匹配后验证 |
| 吸收不预测MCC | 中 | 更大样本量 |
| Matryoshka嵌套无影响 | 中 | 更多消融变体 |
