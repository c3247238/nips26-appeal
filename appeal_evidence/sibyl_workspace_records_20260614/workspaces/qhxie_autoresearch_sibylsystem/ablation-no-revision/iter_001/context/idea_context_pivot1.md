# Pivot Iteration 1 - 基于 Pilot 教训的研究方向

## Pilot 实验关键发现

### 失败的假设
1. **H3 超加性被证伪**: EDW-Step-DPO + 自适应推理 = EDW-Step-DPO 单独使用，21.25% 准确率相同
2. **深度加权无效**: 所有变体的训练损失几乎相同（0.6922-0.6927）
3. **数据集严重倾斜**: 94% L3（概念性）错误，5.7% L1，0% L2
4. **低绝对准确率**: 15-21% vs 预期的 50%+

### 教训
1. **7B 模型在 MATH 上表现差**: 基础准确率应为 50%+，15-21% 表明数据集构建方法有问题
2. **深度加权需要更细致的实现**: 简单的权重乘积不够
3. **Self-Refine 可能不适用于此场景**: 3轮修正后仍有 94% 概念性错误

## 新研究方向要求

1. **高绝对准确率**: 确保基础方法在 benchmark 上能达到合理水平
2. **更小的 pilot**: 先用极小规模验证可行性
3. **与现有工作的清晰区分**: 避免 Uni-DPO、CODA、ODAR、SuperCorrect 的重叠
4. **简单但有效的机制**: 避免过于复杂的加权方案

## 潜在新方向

### 方向 A: 更强的基线 + 简单改进
- 使用更好的基础模型（如 DeepSeek-Math）
- 在标准 Step-DPO 上加微小改进
- 确保基础准确率达到 40%+

### 方向 B: 专注于推理效率
- 不训练模型，而是优化推理策略
- 基于问题特征的动态路由
- 评估指标：准确率/推理成本比

### 方向 C: 混合方法
- 小模型检测问题难度
- 简单问题用小模型，复杂问题用大模型
- 元学习框架

### 方向 D: 错误分析驱动的训练
- 首先分析模型在 MATH 上的具体错误模式
- 针对错误模式设计定向训练
- 而不是通用的深度加权
