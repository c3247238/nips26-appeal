# 理论研究者视角：形式化分析共现方法的局限

## 核心观点

UAD的失败不是偶然的工程问题，而是可以从数学上证明的结构性限制。

## 理论分析

### 1. 吸收的数学定义

设概念C有子概念{c₁, c₂, ..., cₙ}。在SAE中：
- 父特征f_C在C的所有实例上激活
- 子特征f_{cᵢ}仅在cᵢ的实例上激活

**关键观察**：f_{cᵢ}和f_{cⱼ}在token级别互斥：
```
P(f_{cᵢ} > 0 ∧ f_{cⱼ} > 0 | token = cₖ) = 0  (当i≠j≠k)
```

### 2. 共现聚类的假设违反

UAD的共现矩阵M满足：
```
M_{ij} = P(fᵢ > 0 ∧ fⱼ > 0)
```

对于吸收特征：
```
M_{ij} ≈ 0  (对所有i≠j)
```

这意味着吸收特征在共现矩阵中表现为**不相关**而非**聚类**。

### 3. 形式化命题

**命题**：对于任何层级结构H，如果子概念在token级别互斥，则基于共现的聚类方法无法以高于随机基线的准确率检测吸收关系。

**证明概要**：
1. 共现矩阵中吸收特征的相似度≈0
2. 聚类算法会将它们分配到不同簇
3. 因此TPR = P(GT对在同簇) ≈ 1/k（k为簇数）
4. 这与随机基线相同

## 理论贡献

1. **形式化定义**了吸收的数学条件
2. **证明了**共现方法的不可行性
3. **指出了**可行方法的理论基础（语义几何而非共现统计）
