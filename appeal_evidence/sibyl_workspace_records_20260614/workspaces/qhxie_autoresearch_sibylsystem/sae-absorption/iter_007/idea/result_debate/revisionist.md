# 修正主义者分析：基于实验结果的假设修正

## 角色声明

作为修正主义者，我的职责是基于实验证据反思原有假设，识别需要根本性修正的认知框架，并提出新的理论方向。本分析覆盖 Iteration 9 的全部实验结果，包括 Gate 0（零 GPU 分析）和 Gate 1（三项关键实验）的新数据。

---

## 一、需要根本性修正的假设

### 1.1 "Feature Absorption 是竞争排斥现象" -- 必须放弃

**原假设（H2）**：在高稀疏度（L0=22）下，>80% 的检测到的假阴性是层级驱动的竞争排斥（competitive exclusion）。

**实验证据要求彻底修正**：

- **混杂分解**：98.6%（宽松定义）的 FN 是 hedging，仅 1.4%（9/657）是层级驱动
- **收紧 hedging 分类（新数据）**：严格 hedging 率仅 6.2%（41/656），即 93.8% 的 FN 在 L0=176 时其 5 个 parent-associated latent 中没有任何一个激活。这意味着宽松定义的 98.6% 是上界，而真实的 hedging 率可能远低于此
- **激活修补（新数据，决定性证据）**：对 8 个持续核心词执行激活修补，0/8 显示 parent 恢复。主方法（decode-reencode）、残差方法（subtract decoder）、全子特征置零 -- 三种方法均为 0 恢复。控制组同样为 0/65

**修正方向**：Feature absorption 在 JumpReLU SAE 上不是竞争排斥现象。原有的 "parent 被 child 抑制" 的因果叙事在 Gemma 2 2B + JumpReLU 架构上没有实验支持。所谓 "吸收" 更准确地描述为：在低 L0 操作点下，SAE 的稀疏编码选择了不同的特征来表征相同的信息，parent latent 从未被 "竞争" 掉 -- 它根本就没有被编码器选中。

### 1.2 "CMI 可以预测吸收易感性" -- 必须大幅降级

**原假设（H4/H10）**：条件互信息 I(X; f_parent | f_child) 可以在统计上显著地区分被吸收和未被吸收的字母。

**多重证据链条表明该假设不可靠**：

1. **L0=82 的原始结果**：rho=-0.383, p=0.059（未校正），Bonferroni p=0.236。方向一致但不显著
2. **控制 probe F1 后（新数据）**：偏相关 rho=-0.328, 置换 p=0.118, Bonferroni p=0.472。关联进一步减弱
3. **限制于高质量 probe（F1>0.85）的子集**：仅 10 个字母，rho=-0.113, p=0.757。关联几乎消失
4. **L0=22 复制（新数据，决定性证据）**：消除了 probe 质量混杂（所有 25 个 probe 均 F1=1.0）后，rho=+0.044, p=0.835。关联完全消失，甚至方向反转

**修正方向**：L0=82 处观测到的 CMI-absorption 负相关是 probe 质量混杂的伪相关（rho(absorption, F1)=-0.692, p=0.0001）。当这个混杂被彻底消除（L0=22, 所有 F1=1.0），信号归零。CMI 诊断不应作为论文的理论支柱，应降级为"探索性分析，未通过复制验证"。

### 1.3 "跨领域吸收率可以独立解读" -- 必须放弃

**原假设（H5）**：至少 2/4 知识层级领域的吸收率 >10%。

**证据**：所有 5 个领域的测量吸收率均低于其 shuffled 控制。控制失败诊断（新数据）揭示了结构性原因：在 R^2304 的 16384 维解码器空间中，cosine>=0.025 阈值下，随机方向可以匹配 3765 个候选特征（23.0%），与真实 probe 的 3478（21.2%）和 shuffled probe 的 3502（21.4%）无统计差异。P(至少 1 个候选激活) = 1.0。

**修正方向**：跨领域测量的绝对吸收率不可解读。它们应被报告为"度量失效的进一步证据"而非"跨领域吸收率"。

## 二、得到强化的假设（及其修正后的理解）

### 2.1 H1 度量不可迁移 -- 从 "不可迁移" 修正为 "结构性失效"

原假设只说度量 "不迁移"。新的控制失败诊断提供了完整的机制性解释：

- **候选爆炸**：cosine>=0.025 在 2304 维空间中是近乎无意义的筛选条件，23% 的特征通过
- **重叠保证**：L0=82 时，每个 token 约 82 个活跃特征，其中 ~18 个是候选，P(>=1)=1.0
- **死特征膨胀**：18.76% 的特征是死特征，贡献候选计数但永不激活，系统性引入噪声
- **度量退化**：吸收度量在标准阈值下退化为 P(假阴性) x P(任意候选激活) ≈ P(假阴性)

**修正**：这不是简单的 "阈值需要重新校准"。阈值敏感性分析（新数据）证明，即使在最严格的阈值（cosine>=0.05, gap>=2.0），控制失败依然存在（measured 0.118 vs shuffled 0.746, ratio 6.3x）。控制失败在全部 20 个阈值组合中普遍存在（CV=0.077）。这是高维空间中度量的固有限制，不是参数调优可以解决的问题。

### 2.2 H3 L0 相变 -- 从 "有趣的经验发现" 修正为 "核心贡献"

L0 相变（42.85% -> 37.49% -> 14.39% -> 0.84%）是论文最稳健的发现：

- 跨层稳定：L10 13.88%, L12 15.96%, L20 13.55% (CV<10%)
- Spearman rho_s = -1.0（完美单调）
- 相变发生在 L0=40-80 区间，符合预测

**修正**：考虑到 H2 的彻底否定和激活修补的零恢复结果，L0 相变的意义需要重新诠释。它不是 "竞争排斥随稀疏度缓解" 的证据 -- 因为竞争排斥在所有 L0 水平上都不存在。更准确的解释是：L0 相变反映的是 **SAE 编码器在不同容量约束下的特征选择行为变化**。在低 L0 下，编码器被迫使用更少的特征来表征信息，因此更可能选择 "语义内容特征" 而非 "表面属性特征"（如首字母）。随着 L0 增加，编码器有足够容量同时编码两类信息。

## 三、新出现的关键发现及其理论含义

### 3.1 严格 hedging 率的反直觉分布

收紧 hedging 分析揭示了一个令人惊讶的模式：

| 字母 | FN 数 | 严格 hedging 率 | 特点 |
|------|-------|----------------|------|
| G | 21 | 90.5% (19/21) | 唯一高严格率字母 |
| V | 9 | 22.2% (2/9) | 较高 |
| A | 45 | 13.3% (6/45) | 中等 |
| L | 43 | 11.6% (5/43) | 中等 |
| B, D, F, H, J, K, M, N, O, P, R, T, W | 各异 | 0.0% | 15 个字母完全没有严格 hedging |

**修正要求**：G 字母的异常（90.5% vs 其他字母大多 0%）需要个案分析。G 的 parent latent 中 latent #12399 有最高权重（0.607），而 G 在 L0=82 时的吸收率仅 3.12%（所有字母中最低之一）。这表明 G 的 parent latent 在较低 L0 也保持激活，是真正的 hedging 而非度量伪迹。其他字母的 0% 严格 hedging 率意味着绝大多数 "被吸收" 的 token 在 L0=176 时也没有任何 parent-associated latent 激活 -- 这些 token 的首字母信息可能根本没有通过 k-sparse probe 定义的方向进行编码。

### 3.2 Probe 质量是一切的混杂

多个分析独立收敛于同一结论：

- rho(absorption, probe_F1) = -0.692 (p=0.0001) -- L0=82 处的 CMI 分析
- rho(absorption, probe_F1) = -0.762 (p=0.000009) -- 阈值敏感性分析
- L0=22（所有 F1=1.0）时 CMI 相关消失（rho=+0.044）

**修正要求**：论文中所有基于 L0=82 的 letter-level 分析都受 probe 质量混杂影响。排名稳定性虽高（Kendall tau=0.977），但这反映的可能是 probe 质量排名的稳定性，而非真实吸收排名的稳定性。高吸收字母（S 42.8%, C 42.2%, I 37.6%）恰好是 probe F1 最低的字母（S 0.33, C 0.13, I 0.38），而零吸收字母（B, J, K, M, Q, V, Y, Z）大多有高 F1。

### 3.3 8 个持续核心词的本质

激活修补的零恢复率（0/8, 三种方法一致）消除了 "核心词是竞争排斥残余" 的可能性。这些词（eight, liked, lower, offer, often, other, under, until）有共同特征：

- 都是高频英语词汇
- 它们的首字母信息可能从未被模型的该层以可用 SAE latent 解码的方式编码
- 它们的 child feature 有高激活值（13-44 范围），但 child 与 probe 方向的 cosine 仅 0.07-0.25

**修正要求**：这些词应被重新分类为 "编码缺失"（encoding absence）而非 "吸收"（absorption）。首字母信息可能在模型的其他层或以非线性方式编码，不在当前 SAE 层的线性子空间中。

## 四、论文叙事的重大修正

### 4.1 标题/摘要修正

当前标题 "Auditing SAE Feature Absorption: Hedging Dominance, Rate-Distortion Diagnostics, and the L0 Phase Transition on JumpReLU SAEs" 包含三个需要修正的元素：

1. **"Hedging Dominance"**：收紧 hedging 实验表明 98.6% 是宽松上界，严格率仅 6.2%。应修正为 "Metric Miscalibration" 或类似表述，避免暗示 hedging 是已确认的主导机制
2. **"Rate-Distortion Diagnostics"**：CMI 在 L0=22 复制失败（rho=+0.044），理论支柱不再成立。应删除或降级为次要发现
3. **"L0 Phase Transition"**：保留，这是最稳健的发现

建议修正标题方向：**"Auditing SAE Feature Absorption Metrics: Universal Control Failure, Encoding Absence, and the L0 Sparsity Transition"**

### 4.2 贡献声明修正

| 原始贡献 | 修正状态 | 说明 |
|---------|---------|------|
| 首次 JumpReLU 度量审计 | 保留并强化 | 控制失败诊断提供了完整的机制解释 |
| 首次混杂分解（98.6% hedging） | 需重大修正 | 98.6% 是宽松上界，严格率仅 6.2%。"hedging dominance" 叙事过度简化 |
| L0 相变 | 保留并重新诠释 | 不再是 "竞争排斥缓解" 而是 "编码容量效应" |
| 首次跨领域吸收测量 | 降级 | 仅作为控制失败的额外证据，不是独立贡献 |
| CMI 诊断 | 必须大幅降级 | L0=22 复制失败。从 "理论支柱" 降级为 "探索性负结果" |
| 诚实的负结果 | 强化 | 4/7 → 可能 5/7 或 6/7 假设被否定，取决于 H4 的最终分类 |

### 4.3 "两种解释" 段落的修正

原计划在 Section 7.2 加入 "两种解释"（竞争排斥 vs 度量失效）。激活修补结果消除了歧义：

- **竞争排斥**：0/8 恢复，三种方法一致为零。此解释被实验排除
- **度量失效 + 编码缺失**：与所有证据一致。度量的 cosine 阈值在高维空间中近乎无效，而 "被吸收" 的 token 实际上从未在该层的 SAE 表示中编码首字母信息

不再需要 "两种解释" -- 证据收敛于单一解释。

## 五、对下一步行动的修正建议

### 5.1 必须立即执行的修正

1. **CMI 叙事全面降级**：论文中 7+ 处 CMI 过度声称必须修正。L0=22 复制结果（rho=+0.044）应在 CMI 的每次引用中明确提及
2. **hedging 数字修正**：从 "98.6% hedging" 修改为 "严格 hedging 6.2% vs 宽松 hedging 98.6%（上界）"，以严格率为主要报告数字
3. **激活修补结果整合**：0/8 恢复率是最强的因果证据，应在 Section 4.2 中突出展示
4. **控制失败诊断作为新 Section**：候选爆炸的机制解释（23% 候选率、P(>=1)=1.0）应提升为独立小节

### 5.2 理论框架修正

原有的 "successive refinement → CMI 预测吸收易感性" 框架已不成立。替代框架：

**SAE 编码容量理论**：
- 在给定 L0 约束下，SAE 编码器面临特征选择问题
- 高语义价值的特征（如 "常用词义"）优先于低信息量的表面属性（如 "首字母"）
- L0 增加 → 编码容量增加 → 表面属性特征可以被同时编码
- 相变发生在编码容量跨越 "表面属性最低编码成本" 的阈值处

这一框架与所有观测一致：
- L0 相变（编码容量效应）
- 0 恢复率（不是竞争排斥，是编码缺失）
- G 字母的异常（G 的 parent latent 可能编码了除首字母外的其他信息，因此在低 L0 也激活）
- Probe 质量混杂（F1 反映 probe 能否从 SAE 表示中线性解码首字母信息 -- 低 F1 意味着信息本就微弱）

### 5.3 诚实报告的重要性

本项目 7 个原始假设中，2 个确认（H1, H3）、1 个部分支持但已实质性削弱（H4）、4 个否定（H2, H5, H6, H7）。新增的 3 个假设（H8, H9, H10）中：

- H8（激活修补 7+/9 恢复）：**否定**（0/8 恢复）
- H9（严格 hedging >80%）：**否定**（严格率仅 6.2%）
- H10（L0=22 CMI rho<-0.3）：**否定**（rho=+0.044）

总计 10 个假设中 2 个确认、1 个部分支持、7 个否定。这个 70% 否定率本身就是有价值的科学贡献 -- 它表明 SAE feature absorption 的现有理解框架存在系统性偏差。

## 结论

本轮实验结果要求对论文的核心叙事进行根本性重构。"Feature absorption 是竞争排斥" 的框架被激活修补实验彻底否定。"CMI 预测吸收" 的理论支柱被 L0=22 复制实验否定。"98.6% hedging" 的标题数字需要用 "严格 6.2%" 进行限定。

但这些负结果恰恰是论文最有价值的部分。SAE 吸收领域的整个缓解工程（Matryoshka SAE、OrtSAE、ATM-SAE、Masked Regularization）建立在一个从未被严格验证的度量上。本文提供了第一组系统性证据表明：（1）该度量在主流 SAE 架构上结构性失效；（2）被度量检测到的 "吸收" 在因果层面上不是竞争排斥；（3）L0 操作点而非架构是控制吸收表象的关键变量。
