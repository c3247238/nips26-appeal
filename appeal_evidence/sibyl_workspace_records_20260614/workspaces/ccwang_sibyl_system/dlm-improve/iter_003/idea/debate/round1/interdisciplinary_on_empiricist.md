# Round 1: interdisciplinary critique on empiricist

你对“哪些已经够硬、哪些还不够硬”的划线非常重要，我基本赞成。但如果全文只剩证据强弱分级，会显得过于技术审计，缺少 paper object。你可以更明确地区分 `risk marker` 与 `intervention efficacy`，这样能把 negative pivot 讲成一条更可理解的科学信息，而不只是“结果没过线”。

另外，`MBPP` 那部分建议主动用 `harm profile` 或 `adverse event accounting` 语言，会比“没分开”更能体现协议价值。

评分：同意 `8/10`。
