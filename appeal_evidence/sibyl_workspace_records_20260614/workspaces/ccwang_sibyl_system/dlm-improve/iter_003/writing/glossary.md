# Glossary

| Term | Working definition |
| --- | --- |
| Audited negative case | A study in which an apparently positive signal is reinterpreted under stronger controls and written up as a bounded negative result rather than a new method claim. |
| Audited slice | The current-only bundle of 100 audited examples, split evenly across GSM8K and MBPP. |
| Compute-matched active control | `DNB-84`, the arm that matches revision budget more closely than `DNB-64` but does not use the audited intervention. |
| Sham control | `RAND-84`, the budget-matched random-targeting arm used to test whether the apparent signal survives a stronger negative control. |
| Current-only artifact closure | The requirement that all paper claims map to joinable artifacts created inside the active workspace. |
| Sample-level audit | Per-sample accounting of fixed, harmed, unchanged-correct, and unchanged-wrong outcomes. |
| Risk marker | A descriptive signal that helps localize vulnerable samples without validating a causal intervention rule. |
| Localized signal | A bounded empirical advantage on a particular audited comparison, here `CARD-84` over `DNB-84` on GSM8K. |
| Harm profile | The task-specific distribution of failure modes used to localize where an intervention does or does not help. |
| Claim ceiling | The strongest wording the current evidence can support without overstating scope or mechanism. |
