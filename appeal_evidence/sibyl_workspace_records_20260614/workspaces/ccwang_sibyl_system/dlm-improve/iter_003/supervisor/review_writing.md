## Summary

The manuscript is now coherent, honest, and reviewer-aware. Its central empirical logic is sound within the stated scope: `CARD-84` shows a localized signal over `DNB-84`, but that signal does not survive a stronger sham-control test against `RAND-84`. This makes the paper scientifically useful as a negative-case study. It does not make it a broad method or framework paper.

3. The final package is now clear enough to support pipeline continuation rather than another conceptual rewrite.

3. Reproducibility packaging is close, but still missing an explicit release or rebuild statement.
4. Minor LaTeX polish and bibliography strengthening remain worthwhile before treating the package as fully finished.

Advance the pipeline, but use reflection to tighten scope language, strengthen related-work positioning, add an artifact-release note, and finish small typesetting cleanup. The right next move is polishing a narrow negative-case paper, not reopening the scientific mainline.
