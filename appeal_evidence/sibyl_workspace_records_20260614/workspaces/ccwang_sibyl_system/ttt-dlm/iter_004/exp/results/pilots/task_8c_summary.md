### LoRA 轨迹分析

| 指标 | 值 |
|------|-----|
| 最大 delta norm | 2.92e-05 |
| 最大 B norm | 2.88e-04 |
| 平均 DTA 更新次数 | 102.9 / 128 步 |
| 平均 DTA loss | 0.124 |
| 全部收敛 | 16/16 (100%) |

**观察**：LoRA 参数范数极小（delta norm ~1e-5, B norm ~1e-4），说明 DTA 更新非常保守。结合 gamma=0.95 的衰减，参数在每步后迅速收缩到接近零。这解释了为什么 DTA 不会引入退化——更新幅度极小。

## 注意事项

1. **准确率为 0%**：本次使用了 per-sample 独立种子（SEED + idx），导致随机数序列与 task_5a 不同，因此准确率与 task_5a 的 12.5% 不可直接对比。16 样本的 Countdown 准确率本身波动很大。
2. **LoRA 范数极小**：delta norm 仅 ~1e-5，说明 DTA 更新在 bfloat16 精度下可能接近数值下限。建议在 full-scale 实验中考虑是否需要调整 lr 或 gamma 以获得更大的参数更新。
3. **计算开销**：DTA 约 5x vanilla（18.8s vs 3.7s），与 task_2a pilot 的 4.3x 基本一致。
