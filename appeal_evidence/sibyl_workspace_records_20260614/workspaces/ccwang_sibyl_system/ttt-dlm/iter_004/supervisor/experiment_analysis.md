# 实验结果分析

## 实验结果概要

第 4 轮迭代测试了 BSD（信念状态扩散）、RACFG（推理感知 CFG）、A-CFG（自适应 CFG）及其组合，在 n=16 pilot 规模上完成评估。同时 Countdown-500 全规模数据（iter 3 方法）提供了对比基准。

**核心指标汇总**：

| 方法 | Countdown 准确率 | GSM8K 准确率 | FLOPs | 数据规模 |
|------|-----------------|-------------|-------|---------|
| Vanilla | 4.7% (full) / 0-6.2% (pilot) | 25.0% (pilot) / 29.6% (1300/1319) | 1.0x | 全规模+pilot |
| **DMI** | **9.3%** (full, 3-seed) | 25.0% (pilot) | ~1.05x | 全规模验证 |
| SCP | 9.1% (full) | - | ~1.5x | 全规模 |
| BSD (k=0.75) | 6.2% (pilot, n=16) | 18.8% (pilot) | ~1.1x | 仅 pilot |
| **A-CFG (w=1.5)** | **12.5%** (pilot, n=16) | **37.5%** (pilot, n=16) | ~2.0x | 仅 pilot |
| RACFG (JSD) | 0.0% (pilot) | - | ~1.8x | 仅 pilot |
| BSD+A-CFG | 6.2% (pilot, n=16) | - | ~2.1x | 仅 pilot |
| DTA | 4.8% (full) | - | ~2.5x | 全规模 |
| ReMDM-conf | 4.4% (full) | - | ~1.5x | 全规模 |
| RCR | 5.7% (full) | - | ~2.5x | 全规模 |

**关键发现**：
1. A-CFG 在 GSM8K pilot 上 +12.5pp（37.5% vs 25.0%），是四轮迭代中最大的绝对提升
2. RACFG（原创核心：JSD 跨步稳定性引导）完全失败（0%），H5 被证伪
3. BSD+A-CFG 组合无协同效应（6.2% < 各自独立 12.5%），H7 被证伪
4. DMI 仍是唯一经全规模三 seed 验证有效的方法（9.3% vs vanilla 4.7%）

## 各方观点总结

- **乐观者**：A-CFG 在 GSM8K 上 +12.5pp 是项目最佳结果；BSD 虽准确率有限但信念熵分析和文本质量提升有独立价值；四轮迭代构建了完整方法论谱系；论文素材充足，发表路径清晰（88% 可发表概率）

- **怀疑论者**：所有 iter 4 新方法仅在 n=16 上测试，零项达到统计显著性；iter 3 已有教训——pilot 阶段 -16%~-25% 的 PPL 改善在全规模上全部消失；vanilla 在 n=16 上从 0% 到 18.8% 波动说明 pilot 结果本质是噪声；提案的三个支柱（BSD、RACFG、组合协同）全部未通过验证；compute-fair 对比中 vanilla 加步是 Pareto 最优

- **战略家**：A-CFG 全规模验证是必须完成的（成本仅 ~18 GPU·h）；DMI 9.3% 全规模数据提供安全网；建议从"BSD+RACFG 三层架构"重构叙事为"A-CFG + DMI 实证发现"；制定了四个场景的论文策略；PROCEED 但附带明确的 pivot 触发条件

## 分析

### 1. 方法可行性

**原提案核心方法的状态**：
- **BSD**：机制可行（信念熵确实单调递减），但准确率提升有限（6.2%，不优于 DMI）。连续信念表示未能转化为推理能力改善——怀疑论者正确指出 EMA+温度退火在数学上必然导致熵递减，这是平凡发现。
- **RACFG**：核心假设（跨步 JSD 不稳定性标识推理关键位置）被彻底证伪。Dream-7B 的跨步概率分布极度稳定（JSD stability ~0.997），根本无法产生有区分度的信号。这是模型架构层面的限制，非超参数问题。
- **A-CFG**：非原创方法（复现已发表的 NeurIPS 2025 论文），但在 Dream-7B 上展现出强信号，尤其 GSM8K pilot +12.5pp。

**结论**：提案的两个原创核心（BSD 表示层、RACFG 预测层）均未达到预期。真正有效的是对已有方法 A-CFG 的复现。

### 2. 性能表现

**经全规模验证的正面结果**：仅 DMI（9.3% vs vanilla 4.7%，Countdown-500，3 seeds）。

**pilot 阶段的正面信号**：A-CFG 在 GSM8K 上 37.5% vs vanilla 25%，在 Countdown 上 12.5% vs vanilla 0-6.2%。但 n=16 的统计功效极低——iter 3 已有前车之鉴（pilot 大幅改善在全规模上完全消失）。

**负面结果**：RACFG 0%、BSD+A-CFG 组合退化、DTA 低于 vanilla、所有重遮蔽方法无效。

### 3. 改进空间

A-CFG 全规模验证是明确的下一步（~18 GPU·h）。如果 A-CFG 全规模成立：
- 可与 DMI 组合探索正交互补性
- 可扩展到 MBPP 第三 benchmark
- 有文献背书（LLaDA-8B 上 GSM8K 73.5）

如果 A-CFG 全规模也失败：DMI 独立成果仍可支撑论文，加上系统性负面结果分析。

### 4. 时间成本

A-CFG 全规模验证仅需 ~12h 墙钟时间（4 GPU 并行）。这是极低成本的决策性实验——无论结果如何都能锁定论文定位。继续比重新开始高效得多。

### 5. 怀疑论者的批评是否致命？

怀疑论者的批评**严厉但不致命**：
- "n=16 不可信" — **正确且严重**，但解决方案是全规模验证而非放弃
- "compute-fair 比较中 vanilla 最优" — 仅在 n=16 上成立，DMI 在全规模上已证明 1.05x FLOPs 可超越 vanilla
- "提案三个支柱全部未验证" — **正确**，但项目已自然转向 A-CFG+DMI 双核心
- "考虑转为负面结果论文" — 如果 A-CFG 全规模也失败则是合理路径，但在验证之前过早

怀疑论者最核心的贡献是提醒：**必须完成全规模验证才能做任何声明**。这直接决定了下一步行动。

## 决策理由

**PROCEED 的理由**：

1. **DMI 安全网牢固**：9.3% vs vanilla 4.7%，全规模三 seed 验证，计算开销几乎为零。无论 A-CFG 全规模结果如何，这已是可发表的核心贡献。

2. **A-CFG 值得全规模验证**：GSM8K pilot +12.5pp 是整个项目最强信号，且有独立文献验证（A-CFG 论文在 LLaDA-8B 上 GSM8K 73.5）。验证成本极低（~18 GPU·h / ~5h 墙钟），放弃验证是不负责任的。

3. **叙事已自然重构**：从"BSD+RACFG 三层架构"转向"A-CFG + DMI 双核心 + 系统性方法论对比"。这个新叙事更务实、更可信，且有充足的实验数据支撑（7+ 方法全规模对比）。

4. **四个场景均有发表路径**：
   - A-CFG 全规模成功 → NeurIPS/ICLR 主会
   - A-CFG 仅 GSM8K 有效 → ICLR（任务依赖性发现）
   - A-CFG 全面失败 → EMNLP（DMI 核心 + 系统性研究）
   - DMI+A-CFG 组合有效 → NeurIPS/ICML 最强叙事

5. **Pivot 的时机尚未到来**：pivot 的触发条件是"A-CFG Countdown-500 seed 42 < 6%"或"DMI 统计不显著"，这些条件需要全规模数据才能判断，而全规模数据尚在进行中。

**不 Pivot 的理由**：
- 核心假设虽部分被证伪（RACFG、BSD+A-CFG 组合），但 DMI 和 A-CFG pilot 的正面信号为 PROCEED 提供了充分基础
- 备选方案 1（Tolerator+BSD）和备选方案 2（HEX+RACFG）的成功概率（50%/45%）不高于当前路径
- 备选方案 3（系统研究论文）不需要 pivot——即使 PROCEED 后 A-CFG 失败，仍可自然降级到此路径

DECISION: PROCEED
