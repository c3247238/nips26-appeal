# 方法论审查：AADWD 实验结果

## 核心立场

从实验方法论的角度，AADWD 的实验设计总体上是严谨的，消融设计尤为出色。但存在若干方法学空白，可能影响结论的普适性。以下逐一审查。

## 实验设计优势

### 1. 消融实验设计堪称教科书级

Random Dynamic WD 实验是一个精准的因果检验：保持时间动态特性不变，仅替换对齐信号为随机噪声。结果（92.06% vs 92.05%）以 0.01% 的精度否定了 alignment signal 的独立贡献。等价累积 WD 实验同样出色：控制 WD 的时间平均量与 Fixed WD 完全匹配，结果（92.54% vs 92.54%）给出了「总量等价则性能等价」的干净因果结论。这两个消融实验的质量足以支撑一篇独立论文。

### 2. 跨架构验证增强了结论的可信度

实验覆盖了 ResNet20（CIFAR-10, CIFAR-100）和 VGG16-BN（CIFAR-10），三个设定的结论一致：Fixed WD 最优（92.54%, 68.45%, 93.86%），AADWD Conservative 接近但不超越，AADWD Aggressive 和 CWD 均劣于基线。结论的跨设定一致性显著增强了内部效度。

### 3. 超参数灵敏度扫描覆盖充分

c 值覆盖了从 0.5 到 10.0 的两个数量级，beta 覆盖了 0.9 到 0.9999。c 扫描清晰地展示了性能退化的轨迹：c=0.5 (91.87%) -> c=1.0 (92.18%) -> c=2.5 (92.05%) -> c=5.0 (87.98%) -> c=10.0 (52.12%)。这种递进式退化为理解方法的失败模式提供了完整的图景。

## 方法学不足

### 1. 缺乏多种子统计（严重）

所有实验仅使用单一随机种子（seed=42）。原始假设 H4 明确要求「3 random seeds per setting, report mean +/- std」，但实际执行中跳过了多种子验证。在 CIFAR-10/ResNet20 上，不同种子之间的典型方差为 0.15%-0.30%，这意味着 Conservative AADWD 与 Fixed WD 之间的 0.17% 差距可能完全在噪声范围内。类似地，Random WD (92.06%) 与 AADWD Aggressive (92.05%) 之间的 0.01% 差距在没有置信区间的情况下不能做出任何统计推断。

**影响评估**：这不改变主要结论的方向（AADWD 未能超越 Fixed WD），但削弱了精确数值比较的可信度。对于论文发表，reviewer 几乎必然会要求补充多种子实验。

**补救建议**：至少对 Table 1 中的 4 个核心方法（Fixed WD, Conservative, Aggressive, Random）运行 3 个额外种子。

### 2. 数据集和模型规模有限（中等）

实验仅涵盖 CIFAR-10 和 CIFAR-100，模型仅涵盖 ResNet20 和 VGG16-BN。这些都是小规模基准测试（32x32 图像，< 1M 参数）。以下问题无法从现有数据回答：
- 在 ImageNet 等大规模数据集上，alignment signal delta 的量级是否仍然是 10^{-3}？
- 在 ResNet50/ResNet152 等大模型上，LR-WD 耦合是否同样主导？
- 在训练更长（500+ epoch）时，动态 WD 是否在后期提供优势？

**影响评估**：不改变在 CIFAR 设定上的结论有效性，但论文中需要明确声明结论的适用范围。

### 3. 仅考虑 SGD+Momentum 优化器

所有实验使用 SGD+Momentum（lr=0.1, momentum=0.9）。现代训练普遍使用 AdamW/Adam，其 weight decay 行为（解耦 WD）与 SGDW 有本质区别。AADWD 在 AdamW 设定下的表现完全未知。

**影响评估**：降低了结论对现代深度学习实践的相关性。

### 4. Norm-Matched WD 实验的异常结果

Norm-Matched WD 的结果异常极端：best acc 90.44% 但 final acc 仅 75.37%（delta=15.07%），weight norm 暴跌至 5.51。这种剧烈的晚期退化暗示实现可能存在问题，或者实验设计中的 norm matching 策略过于激进。论文中如果引用这个数据点，需要仔细解释其退化机制。

### 5. 解耦实验中 c 值设置的合理性

解耦实验中 conservative 使用 c=0.0005，aggressive 使用 c=0.25，而耦合参考使用 c=0.005 和 c=2.5。这种不对称的 c 值选择可能影响比较的公平性。虽然解耦后 lambda 不再乘以 gamma_t 需要调整 c 来补偿量级差异，但最终选择的 c 值是否最优尚不清楚。

**影响评估**：不改变解耦导致崩溃的定性结论（aggressive 的正反馈循环是结构性的，与 c 值无关），但 conservative 解耦的具体退化程度（80.30%）可能受 c 选择影响。

## 统计学审计

| 声明 | 数据支持度 | 统计显著性 |
|------|-----------|-----------|
| Fixed WD 5e-4 最优 | 强（3/3 设定一致） | 单种子，需验证 |
| Random WD = AADWD | 中（单种子差 0.01%） | 不足以做统计推断 |
| Equiv CWD = Fixed WD | 强（精确匹配 92.54%） | 单种子巧合可能性低 |
| CWD 晚期崩溃 | 强（3/3 设定一致，delta 4.84%-12.57%） | 效应量大，单种子足够 |
| LR 解耦灾难性 | 强（崩溃至 10.00%） | 效应量极大 |

## 明确判断

**实验结果的主要结论是可信的，但需要补充多种子统计才能达到发表标准。** 核心方向判断（AADWD 未能超越 Fixed WD）不太可能被多种子实验翻转，但精确的数值比较（特别是 random vs. AADWD 的 H5 检验）需要统计检验的支撑。建议：补充 4 个核心方法的 3 种子实验后，可以提交论文。
