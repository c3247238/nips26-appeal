# 迭代1 方法论：多种子验证与Gamma平方耦合分析

## 1. 迭代1目标与背景

迭代0已完成核心实验验证（39个任务），得出三个主要结论：
1. **预算等价性**：固定WD精确等价于动态WD的平均
2. **LR-WD耦合必要性**：解耦WD导致灾难性崩溃（92% → 10%）
3. **对齐信号不可行动性**：随机动态WD与对齐感知WD无显著差异

迭代1专注于**统计学加强和机制细化**：
- 补充多种子实验以得出置信区间（NeurIPS标准）
- 验证gamma^2耦合理论（来自[2512.08217]论文）
- 确保解耦实验的c值一致性

## 2. 实验对象：CIFAR-10 / ResNet-20

为了保持1小时的计算预算，迭代1仅在以下设置上运行：
- **数据集**：CIFAR-10（计算最快）
- **模型**：ResNet-20（轻量级）
- **总epochs**：200（与迭代0一致）
- **批次大小**：128
- **学习率**：0.1，余弦退火到0

## 3. 三个子任务

### 子任务1：多种子验证（改进1A）
**目标**：为主要方法对比（表1）补充4个额外种子，共5个种子

**种子列表**：42, 123, 456, 789, 1010

**方法**：
- fixed_wd (λ=5e-4) — 基准
- aadwd_conservative (c=0.005, beta=0.99)
- aadwd_aggressive (c=2.5, beta=0.99)
- aadwd_square (c=0.005, beta=0.99，λ_t = c·γ_t²·(1-δ_t))
- random_dynamic_wd (c=0.005) — 对照
- budget_equiv (λ = mean(aadwd_aggressive轨迹)) — 等价验证

**输出**：
- 每个方法的测试精度：mean ± 2σ (95% CI)
- 验证：核心差异是否统计显著

### 子任务2：Gamma^2耦合验证（新增）
**目标**：测试WD是否应该按γ²而非γ缩放（基于[2512.08217]）

**理论背景**：
- 标准AADWD使用λ_t = c·γ_t·(1-δ_t)，线性于γ_t
- [2512.08217]论文建议WD应按γ_t²缩放以保持稳定性
- 需要验证这是否对对齐感知的WD也成立

**方法变体**：
1. linear_coupling: λ_t = c·γ_t·(1-δ_t) （原始）
2. square_coupling: λ_t = c·γ_t²·(1-δ_t) （新增）
3. conservative_fixed_wd: λ=5e-4 （对比）

**运行方式**：
- 仅在seed=42上运行（成本最低）
- 记录整个200-epoch的lambda轨迹、weight norm轨迹
- 分析early/mid/late阶段的dynamics差异

### 子任务3：解耦实验c值一致性（改进1B）
**目标**：用与主实验相同的c值重新验证完全解耦的崩溃

**现状问题**：
- 主实验用的c值：aggressive c=2.5, conservative c=0.005
- 之前的解耦实验用的c值更小，可能混淆了因果关系

**重新验证**：
- decoupled_aggressive: λ_t = 0（不再依赖δ_t），使用c=2.5的标准差学习率
- decoupled_conservative: λ_t = 0（不再依赖δ_t），使用c=0.005的标准差学习率
- 验证：崩溃是否仍在92% → 10%量级

**预期结果**：
- 验证解耦本身导致崩溃，与c值无关
- 强化"LR-WD耦合是结构性必要"的论断
